Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. Dengan CSS, Anda dapat mengontrol tata letak, warna, font, dan banyak aspek lainnya dari elemen HTML.

Keunggulan CSS

1. Memudahkan pengaturan tampilan dengan fleksibilitas tinggi.

2. Mengurangi ukuran file HTML dan mempercepat pemuatan halaman.

3. Memisahkan konten dari desain, sehingga lebih mudah untuk mengelola.

Kekurangan CSS

1. Perbedaan tampilan di berbagai browser.

2. Beberapa browser lama mungkin tidak mendukung CSS dengan baik.

Cara Kerja CSS

1. Memuat HTML, browser mengambil dokumen HTML.

2. Mengurai HTML menjadi Document Object Model (DOM) dan memuat CSS.

3. Memuat CSS, termasuk gambar dan video.
